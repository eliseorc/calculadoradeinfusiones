# EcoInforme

Primera versión de una aplicación web de informes ecocardiográficos, sin cuentas de usuario ni almacenamiento de datos. La información ingresada queda únicamente en el navegador mientras se prepara el informe.

## Uso

1. Abrí `index.html` con Chrome, Edge o Safari.
2. Completá el formulario de la izquierda.
3. Revisá y, si hace falta, editá manualmente la descripción y las conclusiones.
4. Elegí **Generar PDF / Imprimir** y, en la ventana de impresión, seleccioná **Guardar como PDF**.

## Próxima personalización recomendable

- Reemplazar las frases automáticas por la redacción definitiva del profesional.
- Añadir logo, matrícula, firma y encabezado del consultorio.
- Incorporar los campos que faltan de la planilla original y validar cada fórmula antes de utilizarla en informes clínicos.

La aplicación no realiza diagnóstico: los textos y cálculos deben ser revisados y validados por el profesional antes de emitir el informe.
